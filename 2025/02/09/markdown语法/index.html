最基本的语法

段落与换行

用空白行区分段落 末尾添加两个或多个空格，按回车键,即可换行 或者可以使用<br>换行

标题语法

注意空格

代码

  • 要将单词或短语表示为代码，请将其包裹在反引号 (`) 中
  • 要表示为代码的单词或短语中包含一个或多个反引号，则可以通过将单词或短语包裹在双反引号(``)中。
  • 要创建代码块，请将代码块的每一行缩进至少四个空格或一个制表符。
  • 围栏代码块，即用三个反引号（```）包裹住你的代码，并在上面的反引号后面表明你的代码语言。

基本语法

字体（强调语法）

引用

  • 多个段落的块引用要为段落之间的空白行添加一个 > 符号
  • 嵌套块引用在要嵌套的段落前添加一个 >> 符号
  • 块引用可以包含其他 Markdown 格式的元素，但并非所有元素都可以使用

列表

有序列表

无序列表

删除线与分割线

要使用删除线，前后使用两个波浪号~~

要使用分割线，在单独一行上使用三个或多个星号 (***)、破折号 (---) 或下划线 (___) ，并且不能包含其他内容。

为了兼容性与美观，请在分隔线的前后均添加空白行。

链接

注意标点为英文标点

图片

地址可以是本地的也可以是网络的

可以为图片添加链接

转义字符

要显示原本用于格式化 Markdown 文档的字符，请在字符前面添加反斜杠字符

进阶语法

表格

使用三个或多个连字符（---）创建每列的标题，并使用管道（|）分隔每列。

或者可以通过在标题行中的连字符的左侧，右侧或两侧添加冒号（:），将列中的文本对齐到左侧，右侧或中心。 | Syntax | Description | Test Text |
| :--- | :----: | ---: |
| Header | Title | Here's this |
| Paragraph| Text | And more |

脚注

当您创建脚注时，带有脚注的上标数字会出现在您添加脚注参考的位置。读者可以单击链接以跳至页面底部的脚注内容。[^1]

[^1]: 您不必在文档末尾添加脚注。您可以将它们放在除列表，块引号和表之类的其他元素之外的任何位置。

任务列表

  • first thing
  • second thing
  • third thing
- [x] first thing
- [ ] second thing
- [ ] third thing
