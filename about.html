About this notebook

This is a small, quiet blog. It lives as a set of plain Markdown files in a folder, rendered by a single-page JavaScript front-end. There is no database, no build step, and no tracking.

The visual language tries to stay out of the way: warm paper tones by day, deep amber nights, glass-panel code blocks, and formulas set with enough space to breathe.

How to read this site

  • Browse by category using the pills on the home page.
  • Toggle light/dark with the button in the top right.
  • Click a [[wiki-link]] to jump between related notes.
